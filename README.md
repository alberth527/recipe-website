# 魔法食譜網站

這是一個使用 React 和 Vite 建立的食譜分享平台，用戶可以瀏覽各種食譜、搜尋喜愛的菜餚、收藏喜歡的食譜，以及分享自己的私房菜。

## 主要功能

- 用戶登入/註冊系統
- 食譜瀏覽與搜尋
- 收藏喜愛的食譜
- 上傳和管理私房菜食譜
- 響應式設計，支援各種裝置

## 技術架構

- 前端框架：React + Vite
- UI 元件庫：Material-UI
- 路由管理：React Router
- 狀態管理：React Hooks

## 開發說明

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

### 建置生產版本

```bash
npm run build
```
